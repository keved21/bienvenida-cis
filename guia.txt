## 1) Repo recomendado para “mouse virtual” (mano → cursor)

✅ **AppajiDheeraj/Virtual-Mouse** (MediaPipe + OpenCV + PyAutoGUI)
Incluye **mover**, **click izq**, **click der**, **doble click**, **drag & drop** y **scroll**. ([GitHub][1])

### Instalación (Windows)

1. Instala Python 3.10+ (ideal 3.10/3.11)
2. En PowerShell (o CMD):

```bash
git clone https://github.com/AppajiDheeraj/Virtual-Mouse.git
cd Virtual-Mouse
pip install opencv-python mediapipe pyautogui numpy
python mouse.py
```

**Gestos (según el README):**

* Índice arriba → mover cursor
* Índice+medio juntos → click izquierdo
* Pulgar+índice juntos → click derecho
* Mano abierta (5 dedos) y subes/bajas → scroll ([GitHub][1])

### Tips para que no falle en demo

* Buena luz frontal (sin sombras duras).
* Cámara fija (trípode o laptop estable).
* Fondo simple (evita posters/manos de otros atrás).
* Usa 720p si va lento.

## 4) Lo clave para que “jueguen con manos”: Gesto → Tecla (Espacio / ↑)

Para el Dino y para un clon tipo Geometry Dash, lo que más sirve es que tu programa detecte un gesto y mande:

* **“salto” = tecla Espacio** (o flecha arriba)
* opcional: “agacharse” = flecha abajo (si tu juego lo soporta)

### La forma más rápida (hack efectivo)

Modificar el `mouse.py` (del repo de Virtual-Mouse) para que cuando detecte “pinch” (pulgar+índice cerca) haga:

```python
import pyautogui, time
# ...
if pinch_detected and time.time() - last_jump > 0.25:
    pyautogui.press('space')   # o 'up'
    last_jump = time.time()
```

**Por qué funciona bien en evento:**

* No dependes de apuntar con el cursor al botón exacto.
* Solo necesitas **un gesto claro** para saltar.
